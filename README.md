## 🚀 Come Iniziare

1. **Clona il repo:**
   ```bash
   git clone https://github.com/tuo-username/esp_py.git
   ```
2. **Configura il Laboratorio (Opzionale):**
   ```bash
   cd Appunti-venv
   pip install -r requirements.txt
   ```
3. **Esegui uno script:**
   ```bash
   python 01_basi/01_input_output.py
   ```
